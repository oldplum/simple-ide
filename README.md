# Simple IDE — 轻量级跨平台代码编辑器与“电子猫娘”互动系统

## 项目概述

**Simple IDE** 是一款使用 **Qt (C++17)** 开发的轻量级跨平台代码编辑器。它旨在为开发者提供流利、智能的文本编辑体验，拥有实时语法高亮、括号匹配、代码折叠等高级编辑器辅助功能。

同时，项目创造性地集成了一个**电子猫娘趣味互动陪伴系统（CatWidget）**。“电子猫娘”具备独立的状态机，可根据用户的实时编码行为（如括号匹配、大量删除代码、静置空闲）变换情绪，并支持手动投喂，极大地缓解了开发者的编码压力。

---

## 核心特性

### 1. 核心编辑器辅助
*   **多标签页管理：** 采用经典的 `QTabWidget` 布局，支持同时打开、编辑和管理多个代码文件。
*   **当前行高亮：** 实时对当前光标所在行进行浅灰色背景着色，提升视觉定位体验。
*   **智能行号栏：** 左侧配有行号显示区，随文档总行数变动自动伸缩宽度，且与文本框垂直滚动保持毫米级同步。
*   **括号匹配高亮：** 基于栈算法对当前光标两侧的大括号 `{}` 进行实时配对高亮，并向电子猫咪发射“开心”信号。
*   **代码块层级折叠：** 自动分析花括号的嵌套深度，在行号栏提供折叠/展开折叠图标，隐藏非当前关注的代码块。
*   **回车自动缩进：** 按下回车时，新的一行会自动继承并对齐上一行的前导缩进。

### 2. 多语言语法高亮
*   支持 **C++**、**Python**、**JSON** 三种编程语言的语法实时高亮。
*   **外部配置扩展：** 词法分析规则存放于外部 JSON 配置文件中，高亮引擎动态读取，易于后续扩展其他编程语言规则。

### 3. 查找与替换
*   提供专属的查找替换悬浮对话框。
*   支持**大小写敏感**、**全词匹配**以及**正则表达式**三种查找检索模式，支持向上查找（查找上一个）与全部替换。

### 4. “电子猫娘”互动系统
*   **四种情绪状态：** 默认（Default）、开心（Happy）、生气（Angry）和饥饿（Hungry），对应不同的 GIF 动图展示。
*   **行为联动状态机：**
    *   **开心：** 当成功匹配大括号 `{}` 时触发。
    *   **生气：** 当用户狂按退格键（Backspace）或删除键（Delete）删除代码时触发。
    *   **饥饿：** 引入时间流失控制，长时间无输入或未投喂时自动扣除饱食度，进入红字警示的“饥饿”状态。
*   **交互投喂：** 支持点击“投喂猫粮”按钮或使用全局快捷键 **`Ctrl + Shift + F`** 进行投喂，重置饥饿倒计时。

*   **构建系统：** qmake

## 项目目录结构

```text
simple_ide/
├── src/                    # .cpp 源代码实现
│   ├── main.cpp            # 程序入口，负责 QApplication 初始化与窗口启动
│   ├── MainWindow.cpp      # 主窗口界面，包含多标签页与文件读写管理
│   ├── CodeEditor.cpp      # 代码编辑器类，包含行号绘制、括号匹配与折叠控制
│   ├── Highlighter.cpp     # 正则语法高亮引擎实现
│   ├── FindReplaceDialog.cpp # 查找与替换对话框实现
│   └── CatWidget.cpp       # “电子猫娘”状态机与交互控件实现
├── include/                # .h 头文件声明
│   ├── MainWindow.h
│   ├── CodeEditor.h
│   ├── Highlighter.h
│   ├── FindReplaceDialog.h
│   └── CatWidget.h
├── resources/              # 静态资源目录
│   └── cat_gifs/           # 存放“电子猫娘”各情绪的 GIF 动图
├── ui/                     # Qt Designer 界面文件目录（暂空，本项目采用纯代码布局）
├── docs/                   # 项目说明文档目录
└── simple_ide.pro          # qmake 项目配置文件
```

## 构建与运行

### 依赖环境
确保您的系统已安装：
*   支持 C++17 的编译器（如 GCC 8+，Clang，或 MSVC）
*   Qt 5.12 或更高版本（包含 `QtWidgets`、`QtGui`、`QtCore`）

### 编译步骤
请在项目根目录下通过终端执行以下命令进行离线构建（**注意：构建产物统一生成于 `build/` 目录中**）：

```bash
# 1. 创建构建目录并进入
mkdir -p build && cd build

# 2. 生成 Makefile
qmake ../simple_ide.pro

# 3. 多线程编译
make -j$(nproc)

# 4. 运行可执行程序
./simple_ide
```
